BOM 浏览器基础模型 主要是操作 浏览器的
DOM 是文档对象模型 D 表示整个网页 O 表示可以操作网页的对象 有属性与方法可以调用 M为文档的树型结构
DOM 树型结构 根节点html 子节点 head body； head 子节点 meta title ;body子节点 有 ul table 等
html节点 又称为html元素
节点分为三类 元素节点 其实就是标签 文本节点 其实就是标签内的文本 属性节点 其实就是标签的属性
查找元素 查找元素节点 document.getElementById(); 参数传递一个元素的id值 这样就获取可以操作该节点的对象
为什么 查找不到元素 这里需要注意 DOM操作必须等待HTML文档加载完毕 解决办法一： 把DOM操作的script标签代码放到html文档最后 解决办法二：用onload 事件 window.onload= function(){ //这里存放当网页加载完毕后 ，再执行的代码块 } id 表示元素节点的唯一性；一个页面的元素节点 id 不能重复； 如果是 IE5.0-一下 可能不兼容 getElementBtId 可以做个判断 非w3c标准 但是兼容性好 建议使用
